# 💬 Tugas Akhir Hackti8 x Maju Bareng AI : LLM-Based Tools And Gemini API Integration For Data Scientics | Chatbot Restoran Untuk Katalog Menu Dan Reservasi Meja

A simple Streamlit app that shows how to build a chatbot using Gemini.
https://restoarsbot.streamlit.app/

[![Open in Streamlit](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://chatbot-template.streamlit.app/)

### How to run it on your own machine

1. Install the requirements

   ```
   $ pip install -r requirements.txt
   ```

2. Run the app

   ```
   $ streamlit run streamlit_app.py
   ```
